## 1. 프로젝트 생성하기
```shell
npm init @vitejs/app
```

```shell
yarn create @vitejs/app
```

```shell
npm init @vitejs/app [폴더명] --template [템플릿]
```

## 2. 빌드 및 데브서버 실행
```json
{
    "dev": "vite",
    "build": "tsc && vite build",
    "serve": "vite preview"
}
```
